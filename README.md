# TheGenWorks - Next-Gen Marketing Website

## Project Description

A modern, AI-powered marketing website built with React, TypeScript, and Tailwind CSS. This project showcases TheGenWorks' services, tech stack, and capabilities.

## Development Setup

Clone this repository and install dependencies to start developing locally.

### Prerequisites

- Node.js (v18 or higher)
- npm, pnpm, or bun

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd genworks-next-glow-main

# Install dependencies (choose one)
npm install
# or
pnpm install
# or
bun install

# Start the development server
npm run dev
# or
pnpm dev
# or
bun dev
```

The development server will start at `http://localhost:8080`

## Tech Stack

This project is built with:

- **Vite** - Lightning-fast build tool
- **React** - UI library
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **shadcn/ui** - Re-usable component library
- **React Router** - Client-side routing

## Deployment

This project can be deployed to any static hosting service:

- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages

Build the project for production:

```sh
npm run build
# or
pnpm build
# or
bun run build
```

The production-ready files will be in the `dist` folder.
